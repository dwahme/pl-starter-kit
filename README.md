
# Welcome to YorkSlang

Data types:
    - Boolin (Boolean)
        - Wavy (True)
        - Wack (False)
        - Whatever (Maybe)

Functions:
    All functions are in polish notation

    (Nah Boolin) (Not Boolean)
    (Yurrd Boolin Boolin) (And Boolean Boolean)
    (Uh Boolin Boolin) (Or Boolean Boolean)
